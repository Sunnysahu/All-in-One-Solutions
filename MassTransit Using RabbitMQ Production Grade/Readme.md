## Overview

This project demonstrates a **production-ready enterprise messaging architecture** built with .NET. It focuses on reliability, scalability, and distributed system patterns.

### Key Features

- Queue-based asynchronous processing
- Reliable messaging with MassTransit
- Retry policies and fault handling
- Dead Letter Queue (DLQ) support
- Distributed workflows using Saga Pattern
- Outbox Pattern for transactional consistency
- Idempotency to prevent duplicate processing
- SQL Server persistence
- RabbitMQ integration
- Structured logging with Serilog
